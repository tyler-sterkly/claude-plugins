# sys-frontend

Design guidance skill for distinctive, intentional visual UI. Provides aesthetic direction, typography approach, and opinionated design choices grounded in the actual subject matter of the brief.

## Design decisions

- Skill is guidance-oriented, not a generator -- it shapes how UI is built rather than producing a specific output
- Central principle: ground choices in the subject's own world (its materials, artifacts, vernacular) rather than reaching for template defaults
- Typography is treated as a primary design element, not a neutral delivery vehicle
- Requires identifying one concrete subject + audience + page job before designing, even if the brief does not pin it down

## Related skills

- `sys-logo`: Logo design (distinct from full UI/page design)
- `sys-svg`: SVG authoring for UI graphics and icons
